## Dependency
```
$ pip install -r requirements.txt
```

## Usage
* Run as CLI
```
$ python3 run_cli.py <user | miner>
```

* Run as GUI
```
$ python3 run_gui.py <user | miner>
```

## CLI Commands
* Join a peer
```
> join <host:port>
```

* Generate a transaction
```
> tx <shorthand_id> <your_message>
```

* Terminate this node
```
> quit
```
